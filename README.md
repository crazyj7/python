Jupyter Key

+셀 모드
    Esc  or ^M ; 셀모드 진입 (헤드가 파란색)
    a ; 위에 셀 추가
    b ; 아래에 셀 추가
    x  or dd ; 삭제
    c ; copy
    p ; paste
    shift m ; 아래셀과 merge
    m ; markdown으로 변경
    y ; code로 변경
    ^ s or s; save
 
 +코드 입력 모드 (edit)
     enter ; 코드 입력로 진입 (헤드가 초록색)
     shift enter ; 선택 셀 실행하고 다음셀로 이동
     ctrl enter ; 선택 셀 실행만.
     alt enter ; 선택 셀 실행하고 다음에 빈셀 추가.
     ctrl shift - ; 현재위치에서 셀을 반으로 나누기
     ctrl / ; 주석처리.
